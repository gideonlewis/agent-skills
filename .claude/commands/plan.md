# /plan

Create an actionable plan from a specification.

**Phase: PLAN**

When invoked:

1. Read `SPEC.md` if it exists
2. Break into atomic tasks (1-4 hours each)
3. Identify dependencies and ordering
4. Prioritize ruthlessly (what's the MVP minimum?)
5. Output `PLAN.md` with task sequence

Each task should be:
- Completable by one person
- Independently reviewable
- Shippable
- Test-verified

Reference the `planning-and-task-breakdown` skill for detailed process.
